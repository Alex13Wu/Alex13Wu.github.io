detachment

2017-11-22

《超脱》在我心中地位永远前三！

“And never have I felt so deeply at one and the same time so detached from myself and so present in the world.”
第一次看《超脱》时还不知道加缪，萨特，存在主义，没看过《1984》，没读过过爱伦坡。这些在里面提到的东西，在二刷三刷四刷的过程中，慢慢进入我的生活。
开头加缪说的话翻译过来是“我从未如此深刻地感受到，我的灵魂离我如此遥远，而我的存在如此真实”。
于是我成了一个存在主义者。
因为这个世界本身就是荒谬的，存在即意义。
不聊电影情节，只说影响。
最直接的影响是大二暑假参加的国际交流生活动中和MIT的学生聊起存在主义哲学有模有样，最装X的举动是他们聊到的书即使在当时我也大部分都看过。临别前James还把他一直随身带的《sense and sensitivity》送给了我，不过现在都还没看。。。
更后面的影响是彻底活在当下，只要大方向把握对了，其他就率性而为。
当然，我只是个废人，但有何不可？
“你看到的只是你想看到的”，大概是当时迫切需要大方向的指导，而存在主义确实适合我的性格，电影推波助澜，把我想要的推到了我身边。
机缘巧合看到一篇的文章，爱不释手，顺藤摸瓜找到了作者，一个在冰岛留学的女生，在豆瓣上和她聊了文中提到的萨特加缪，再后来豆瓣账号丢了，也懒得再找，只是珍藏的文章现在仍是挚爱，时不时翻出来看看，感受依旧深刻！
不久前朋友开玩笑说我写的观后感逻辑有点问题，但这真的不是我在意的东西，正如我从不推荐电影，闲书，写这些的目的也仅仅是记录我深爱的荒废的旧时光，“我爱这哭不出来的浪漫”。
电影结尾的爱伦坡小说的文段被我写在一把扇子上和自己素描的幻想女神一起挂在宿舍的墙上：
“我独自一人策马前行
穿过这片阴沉的，异域的乡间土地
最终，当夜幕缓缓降临的时候
厄舍府冷清的景色展现在我眼前
我未曾目睹过它过往的模样
但仅凭方才的一瞥，某种难以忍受的阴郁便浸透了我的内心
我望着宅邸周围稀疏的景物
围墙荒芜，衰败的树遍体透着白色
我的灵魂失语了
我的心在冷却
下沉”
——爱·伦坡《厄舍府的倒塌》
